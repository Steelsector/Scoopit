# Scoopit

Scoopit is a Chrome extension that helps import Garage61 setups from laps pages into your setups folder with manual start control.

## Features

- Manual start from extension popup
- Imports only valid setup buttons (wrench icon)
- Skips duplicates and handles "already exists"
- Optional dry-run mode
- Configurable max rows, retry, delay, and debug mode

## Install (Local)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder:
   - `Scoopit`

## Package for Sharing

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Pack extension**
4. Choose extension root:
   - `Scoopit`
5. Click **Pack Extension**

Chrome will generate:
- `.crx` package file
- `.pem` key file (keep this safe for future updates)

## Usage

1. Open a Garage61 laps page (`https://garage61.net/app/laps/...`)
2. Click the Scoopit extension icon
3. Configure options in popup if needed
4. Click **Start Import**
5. Check browser console for summary/debug logs
